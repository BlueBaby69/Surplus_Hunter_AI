# 🛰️ Surplus Hunter AI

**Surplus Hunter AI** is a full-stack, AI-powered research and workflow platform designed to discover, manage, and process foreclosure surplus funds and tax sale overages using publicly available government data.

Built for **USA Track & Reclaim LLC**, this system automates lead discovery while maintaining human approval for all outreach and claim actions.

---

## 🚀 Features

### 🔍 Data Discovery
- Multi-source scraping (HTML, PDF, CSV)
- Public government data ingestion
- Scheduled automation (GitHub Actions)

### 🧠 AI Processing
- Data normalization
- Intelligent duplicate detection
- Lead scoring system
- Compliance rule evaluation

### 📋 CRM Pipeline
- Kanban workflow stages
- Lead tracking from discovery → recovery
- Status updates and activity logs

### 📄 Document Generation
- Claim checklist PDFs
- Auto-filled document templates

### 📧 Email Workflow
- AI-generated email drafts
- Human approval required before sending

### 📊 Claim Tracking
- Track submissions, progress, and recoveries
- Revenue tracking system

### 🔐 Authentication & Security
- Supabase Auth integration
- Role-based access control
- Secure API architecture

### ☁️ Cloud Deployment (Free-tier compatible)
- Frontend: Vercel
- Backend: Render
- Database: Supabase
- Automation: GitHub Actions
